# biblioteca_php

Sistema de emprestimo de livro e revista em uma biblioteca com cadastro de aluno e professor. 
Contendo a regra de alunos poderem pegar apenas um livro por vez e professor podendo pegar 3 livros por vez. Esta regra não se aplica a revista que ambos podem pegar multiplas revistas por vez.

O sistema é Feito em POO, contendo as classes Livro, Revista, Usuario, Estante, Bibliotecario, Aluno que extende Usuario e professor que tembém extende Usuario.
A interface "ItemEmprestavel" que implementa Livro e Revista.
